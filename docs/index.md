# CppCmakeProjectTemplate

Professional C++ CMake Project Template

**Version:** 1.2.0

## Libraries

- **dummy_lib** — `libs/dummy_lib/`
- **fuzzable** — `libs/fuzzable/`
- **secure_ops** — `libs/secure_ops/`

## Applications

- **main_app** — `apps/main_app/`
- **demo_app** — `apps/demo_app/`
- **extreme_app** — `apps/extreme_app/`
- **gui_app** — `apps/gui_app/`

## Quick Start

```bash
# Build
python3 scripts/tool.py build

# Run tests
python3 scripts/tool.py build check --no-sync

# Generate docs
python3 scripts/tool.py doc build
```
