# C++ CMake Scaffolder

Professional C++ CMake Project Generator: Sanitizers, Cross-Platform Presets, Embedded Ready.

## Özellikler

- Hızlı C++ CMake proje iskeleti oluşturur (sanitizers, platform preset'leri, embedded-ready).
- Komutlar:
  - `CppTemplate: Create New Project` — proje oluşturur (`cpp-cmake-scaffolder.init`).
  - `CppTemplate: Library Manager (toollib)` — kütüphane yönetimi (`cpp-cmake-scaffolder.toollib`).
  - `CppTemplate: Project Orchestrator (toolsolution)` — proje orkestrasyonu (`cpp-cmake-scaffolder.toolsolution`).

## Hızlı Başlangıç

1. VS Code'da Komut Paletini açın (Ctrl+Shift+P).
2. `CppTemplate: Create New Project` komutunu çalıştırın ve istemleri izleyin.
3. Oluşturulan projede `CMake Tools` veya terminal üzerinden derleyip çalıştırın.

## Gereksinimler

- Visual Studio Code (engines: ^1.80.0)
- Önerilen eklentiler: `C/C++` (ms-vscode.cpptools), `CMake Tools` (ms-vscode.cmake-tools)
