secure_ops
==========

`secure_ops` provides a small set of safety-focused helpers used by the example `extreme_app` and fuzz targets.

It is implemented to compile under the `extreme` hardening profile and aims to demonstrate idiomatic, analyzable, and sanitizer-friendly C++ code.

Contents
- `include/secure_ops/` — public headers
- `src/` — implementation

License and contribution notes are in the top-level `LICENSE` file in this directory.
